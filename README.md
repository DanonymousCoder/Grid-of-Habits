# Grid Of Habits
## A daily habit tracker(with a twist) inspired by the theme "grid".

## Features
* Create/Add Habits
* Streak determination
* Progress monitoring with coloured squares
* Delete Habit
* Light/Dark mode

## Technologies used
* React
* TypeScript 
* CSS
* Lucida Icons
